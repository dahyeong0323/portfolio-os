# Stage 3 CLI Usage

All commands support global `--db`.

## Research

```text
portfolio-os create-thesis
portfolio-os update-thesis-status
portfolio-os add-research-source
portfolio-os list-research-sources
portfolio-os create-research-packet
portfolio-os add-research-fact
portfolio-os add-missing-data
portfolio-os run-research-qa
portfolio-os show-research-packet
portfolio-os export-research-report
```

## Macro Context

```text
portfolio-os build-portfolio-context
portfolio-os show-portfolio-context
portfolio-os record-macro-metric
portfolio-os record-correlation
portfolio-os classify-macro-regime
portfolio-os create-crash-playbook-rule
portfolio-os create-macro-context
portfolio-os validate-macro-context
portfolio-os show-macro-context
portfolio-os export-macro-context-report
```

## Reports

- Research: `data/exports/research_reports/research_packet_<id>.md/.json`
- Macro: `data/exports/macro_reports/macro_context_<id>.md/.json`
