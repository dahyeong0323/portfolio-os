# Stage 4 CLI Usage

All commands support global `--db`.

```text
portfolio-os build-senior-input-bundle
portfolio-os show-senior-input-bundle
portfolio-os create-senior-memo
portfolio-os add-senior-section
portfolio-os add-decision-candidate
portfolio-os add-no-action-alternative
portfolio-os add-opposing-argument
portfolio-os add-decision-change-trigger
portfolio-os run-senior-memo-qa
portfolio-os show-senior-memo
portfolio-os export-senior-memo-report
portfolio-os archive-senior-memo
```

Reports are written to:

```text
data/exports/senior_memos/senior_memo_<id>.md
data/exports/senior_memos/senior_memo_<id>.json
```
